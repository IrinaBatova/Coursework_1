# Проект "Coursework_1"

## Описание:

Проект "Coursework_1" - это итоговая курсовая работа на Python по разделу 3
"Библиотеки для Python разработчика" в рамках обучения по программе 
"Python-разработчик".

## Установка:

1. Клонируйте репозиторий:
```
git clone git@github.com:IrinaBatova/Coursework_1.git
```

2. Все данные о проекте находятся в файле:
```
pyproject.toml
```

## Использование:

1. Запустите файл main.py
2. Введите запрошенные данные.

## Тестирование:

1. Для тестирования используется библиотека pytest
2. Запуск тестирования: введите команду poetry run pytest --cov в терминале.
3. Code coverage 80%.
